Shortest Path

Find the shortest path between any two nodes using (BFS, DFS, Best-FS, Bidirectional-FS, A*) Algorithms. Then compare the results of each algorithm.

------------------------------------------------------------

Technology:

Programming language:
Python 3.0

Used Libraries:
* networkx
* numpy
* queue.PriorityQueue
* sys
* panda
* time

------------------------------------------------------------

Content:

Data:
Two text files that contains the data.
* weighted.txt
* unweighted.txt

Algorithms' Codes:
Each file contains an algorithm that is implemented by a data structure.
Naming these files follows this rule: (algorithm_name_data_structure.py)
* breadth_first_search_queue.py
* breadth_first_search_dictionary.py
* depth_first_search_stack.py
* depth_first_search_array.py
* bidirectional_first_search_queue.py
* bidirectional_first_search_array.py
* best_first_search_priority_queue.py
* best_first_search_minimum_heap.py
* best_first_search_btree.py
* a_star_hash_table.py
* a_star_priority_queue.py

Reading data:
This file has many methods that read and store the data in many ways like adjacency matrix graph and adjacency list graph
* read_data.py

Main:
* main_shortest_path.py

------------------------------------------------------------

Usage:

To run this program, simply sun the main file, choose the number of the algorithm you want to use, then choose the starting and target nodes. The path between these nodes will appear on the terminal.
